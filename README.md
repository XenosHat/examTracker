# Progress Tracker

**Progress Tracker** is a lightweight, single-file study dashboard designed to track syllabus completion, revisions, mock tests, and notes. It is ideal for exam preparation workflows such as SSC CGL, UPSC, or any structured syllabus-based learning.

The application runs entirely in the browser and requires no installation, making it portable, fast, and easy to use.

---

# Advantages / Pros

## 1. No Installation Required
The entire application runs from a single `index.html` file.  
Simply open it in any modern browser and start using it immediately.

## 2. Single-File Application
All logic (HTML, CSS, and JavaScript) is contained in one file.  
This makes the project extremely portable and easy to share.

## 3. Fast and Lightweight
Because there are no heavy frameworks or dependencies, the application loads instantly and runs smoothly even on low-end devices.

## 4. Works Offline
The app can function completely offline using local storage.  
You do not need an internet connection to track your study progress.

## 5. Optional Cloud Sync
If configured with Firebase, the application can synchronize your data across devices.

## 6. Secure Profile System
Profiles are generated using a passphrase-based hash system.  
Your passphrase itself is not stored directly, improving privacy.

## 7. Syllabus Tracking
You can track multiple subjects and topics with progress indicators showing completion status.

Typical status flow:
- Not Started
- In Progress
- Completed

## 8. Topic Progress Percentage
Each topic can store a completion percentage so you can measure how much of a topic you have covered.

## 9. Mock Test Tracking
The app allows you to log mock tests and monitor performance trends.

It tracks:
- Score
- Total Marks
- Date
- Notes

You can easily analyze your improvement over time.

## 10. Revision Counter
Each topic can store revision counts so you know which topics you have revised the most.

## 11. Notes System
You can attach notes to individual topics for quick reference.

## 12. Pin Important Topics
Important subjects or topics can be pinned so they appear at the top of your dashboard.

## 13. Backup and Restore
You can export your entire profile as a JSON file and import it again later.

This allows:
- Device migration
- Data backup
- Easy sharing

## 14. Responsive Design
The interface is optimized for both desktop and mobile devices.

## 15. Simple and Clean UI
The interface is minimal and focused only on productivity features without unnecessary distractions.

## 16. Easy Customization
Since everything is in one HTML file, developers can easily customize:

- UI colors
- Subjects
- Topic structure
- Features

# How to Use

1. Download the project files.
2. Open `index.html` in any browser.
3. Enter a passphrase to create your profile.
4. Start adding subjects, topics, and tracking your progress.
